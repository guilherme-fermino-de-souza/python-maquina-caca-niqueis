# python-maquina-caca-niqueis
Algoritmo de máquina de caça-níqueis feita no python com auxílio do vídeo "Learn Python With This ONE Project!" do canal Tech With Tim. Simula o comportamento de uma máquina de caça-níqueis ao permitir que o usuário insira um saldo inicial e aposte no jogo, retornando o resultado final vitória ou derrota, assim como a quantia ganha.
